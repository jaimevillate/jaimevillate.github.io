Exam creation with Maxima

The function quiz creates an exam in LaTeX format. The structure of the function is:

quiz ("name", [problem 1, problem 2,..], [question 1, question 2,...])

Example with 2 problems and 15 multiple-choice questions, whose source code is in a file questions.mac.

the LaTeX file exam_20150626.tex is created, which can then be automatically converted into the XML format used by Moodle.
