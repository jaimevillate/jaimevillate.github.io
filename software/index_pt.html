A lista seguinte são os pacotes de software que tenho escrito e publicado.

  1. physics.asy. Um módulo que acrescenta a Asymptote algumas funcionalidades úteis na criação do tipo ilustrações encontradas nos livros de física. O manual está disponível em formatos HTML e PDF.
  2. Maxima. Sou um dos programadores responsáveis por este Sistema de Computação Algébrica (CAS). A minha contribuição principal tem sido novas versões dos programas para traçado de gráficos, um módulo de análise gráfica de sistemas dinâmicos, funções para resolução numérica de sistemas de equações diferenciais, um módulo de análise combinatória e um pacote de simulação de circuitos quânticos. Também tenho melhorado a interface gráfica Xmaxima e o sítio Web
  3. psimage. Pacote antigo que criei na década de 1980 para elaborar as figuras da minha tese de doutoramento. Continuei a desenvolvê-lo e usá-lo nos meus livros até a década de 2000, mas agora uso o software Asymptote em vez deste pacote.
  4. parsewiki. Pacote disponível nos repositórios de Debian e Ubuntu. Produz HTML, XHTML, Latex ou DocBook/XML a partir de ficheiros fonte em texto simples, com sintaxe minimalista, do género da sintaxe usada nos sistemas Wiki. Escrevi este script por volta do ano 2000, mas já não estou a usá-lo porque as versões HTML dos meus livros mais recentes são produzidas diretamente a partir das fontes LaTeX.
